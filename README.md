## Objectives
  
- Provide real world data examples and develop insights using publicly available real estate valuation data 
  - Source: Dallas Central Appraisal District (DCAD)
  - Tools: Excel & SQL
  - Deliverables:
  	- Review of production code
		- Audit Verification Process
		- USPAP Reporting
	- Work through examples
		- EVR Audit
		- Single Family Residence Averages
- Discuss what data analysis looks like in local government
  - Importance of domain knowledge
  - Need for strong communication skills
  - Benefits of interdisciplinary skills

## Data in Repository

There are example data in the `Data` folder for the following projects:

AdHoc Report: Homeownership age trends 2003 - 2025
	Simple example of the power of excel pivot table / graph
	
Audit Verification: Production example of 3rd party audit request report automation

EVR Audit: Example QC Analyst process to verify output of a pdf report for Highland Park ISD

Single Family Residence Averages: Example using SMU DCAD database to calculate average values and import to a formatted Excel workbook
